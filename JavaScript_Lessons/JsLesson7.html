JavaScript Lesson: Functions

Learn to code from the very beginning

1. Instructions

Functions are like mini-programs inside your program. They group code together so you can reuse it.

Example: A function that greets someone:

function greet() {
console.log("Hello!");
}

greet();

Step-by-step:

1. "function greet() {" tells JavaScript you are creating a function called greet.

2. The code inside the curly brackets is what the function does.

3. Calling greet() runs the code inside the function.

Guided experiment: Add another console.log inside the function and call it again. What happens?

Takeaway: Functions let you reuse code and organise your program better.
