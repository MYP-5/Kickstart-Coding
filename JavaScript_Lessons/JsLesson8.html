JavaScript Lesson: Lists

Learn to code from the very beginning

1. Instructions

Arrays are like collections of values stored together in one variable.

Example: An array of fruits:

let fruits = ["apple", "banana", "cherry"];
console.log(fruits);

Step-by-step:

1. JavaScript sees the square brackets and stores multiple items in one variable.

2. console.log(fruits) shows all the items in the array.

Guided experiment: Add another fruit to the array and print it. What happens?

Takeaway: Arrays let you store many items together and access them easily.
