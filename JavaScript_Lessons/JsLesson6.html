JavaScript Lesson: Loops

Learn to code from the very beginning

1. Instructions

Loops let your code repeat actions without writing the same code again.

A for loop is used when you want to repeat something for each item in a list.

Type this code into the editor:

let fruits = ["apple", "banana", "cherry"];
for (let fruit of fruits) {
console.log(fruit);
}

Step-by-step:

1. JavaScript looks at the array called fruits.

2. It takes one item at a time from the array.

3. Each item is stored in the variable fruit.

4. console.log(fruit) runs once for each item.

Guided experiment: Add another fruit to the array and run the code again.

Takeaway: for loops let you repeat actions for each item in an array.
