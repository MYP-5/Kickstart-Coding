JavaScript Lesson: Introduction

Learn to code from the very beginning

1. Instructions

Welcome to JavaScript! This is where your coding journey starts.

JavaScript is a programming language. It lets you tell a computer what to do using instructions called code.

Think of JavaScript like a recipe: each line is a step the computer must follow in order.

Type this into the editor:

console.log("Hello, world!");

What JavaScript does:

1. Looks at the first line.

2. Sees the command console.log() and knows it should show text.

3. Displays exactly what's inside the quotation marks in the output area.

key note: Every line must end with a semi-colon

Guided experiment: Change the text to your name and run the code. What do you expect to see?

Takeaway: JavaScript reads instructions line by line and follows them exactly. The console.log() command shows text on the screen.
