JavaScript Lesson: If Statements

Learn to code from the very beginning

1. Instructions

If statements let your code make decisions based on conditions.

Think of it like a choice: if something is true, do one thing, otherwise do something else.

Type this code into the editor:

let age = 15;
if (age < 13) {
console.log("You are a child");
} else if (age < 18) {
console.log("You are a teenager");
} else {
console.log("You are an adult");
}

Step-by-step:

1. JavaScript checks the first if condition.

2. If it is false, it checks the else if condition.

3. If all conditions are false, it runs the else code.

Guided experiment: Change the value of age and predict which message will print before running the code.

Takeaway: if, else if, and else let your program choose between different outcomes.
