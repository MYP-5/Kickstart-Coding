JavaScript Lesson: Final Project

Learn to code from the very beginning

1. Instructions

This final lesson brings together everything you have learned so far.

Your goal is to create a simple program that talks to the user and makes a decision.

Start with this code:

let name = prompt("What is your name?");
let age = parseInt(prompt("How old are you?"));

Add an if statement that:

- Prints one message if the user is under 18

- Prints a different message if they are 18 or older

Optional challenge: Store an array of favourite foods and use a for loop to print them.

Takeaway: This project shows how prompt, variables, if statements, and loops work together.
