JavaScript Lesson: Data Types

Learn to code from the very beginning

1. Instructions

Not all information in JavaScript is the same. Different kinds of values are called data types.

The main types are:

- Text (strings), written inside quotation marks

- Numbers, written without quotation marks

- True or false values (booleans)

Example: Store a name and a number:

let name = "Alex";
let score = 10;

Step-by-step:

1. JavaScript sees "name = 'Alex'" and stores the text in a box called name.

2. JavaScript sees "score = 10" and stores the number in a box called score.

Guided experiment: Print both values using console.log(name) and console.log(score). Try changing them and run the code.

Takeaway: JavaScript uses data types to understand what kind of value it is working with.
