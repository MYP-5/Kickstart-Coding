JavaScript Lesson: Dictionaries

Learn to code from the very beginning

1. Instructions

Objects store information using key-value pairs.

Think of an object like a real dictionary: you look up a word (key) to get a meaning (value).

Type this code into the editor:

let ages = {Alice: 14, Bob: 16};
console.log(ages);

Step-by-step:

1. JavaScript creates an object using curly brackets.

2. Each name is a key, and each number is its value.

3. console.log(ages) shows the whole object, including the curly brackets.

Guided experiment: Add a new name and age inside the object and print it again.

Takeaway: Curly brackets are normal when printing objects. They show all stored key-value pairs.
