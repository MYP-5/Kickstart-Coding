Python Lesson: Input & Output

Learn to code from the very beginning

1. Instructions

So far, your code has only shown information. Now you will learn how to get information from the user.

In Python, the input() function is used to ask the user to type something.

Type this code into the editor:

name = input("What is your name? ")
print(name)

Step-by-step:

1. Python shows the message asking for a name.

2. It waits for the user to type something.

3. Whatever the user types is stored in the variable called name.

4. print(name) shows what the user typed.

Guided experiment: Run the code, type a name, then change the question and try again.

Takeaway: input() lets your program communicate with the user.
