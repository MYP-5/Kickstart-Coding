Python Lesson: Data Types

Learn to code from the very beginning

1. Instructions

Not all information in Python is the same. Different kinds of values are called data types.

The main types are:

- Text (strings), written inside quotation marks

- Numbers (integers and decimals), written without quotation marks

- True or false values (booleans)

Example: Store a name and a number:

name = "Alex"
score = 10

Step-by-step:

1. Python sees "name = 'Alex'" and stores the text in a box called name.

2. Python sees "score = 10" and stores the number in a box called score.

Guided experiment: Print both values using print(name) and print(score). Try changing them and run the code.

Takeaway: Python uses data types to understand what kind of value it is working with.
