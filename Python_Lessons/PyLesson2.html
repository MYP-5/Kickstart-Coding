Python Lesson: Variables

Learn to code from the very beginning

1. Instructions

Variables are like labelled boxes where you can store information to use later.

Think of a variable as a name tag on a box that holds something inside.

Example: Store a number in a variable called age:

age = 5

Step-by-step:

1. Python sees "age = 5".

2. It creates a box called "age".

3. It puts the number 5 inside that box.

Guided experiment: Print the variable:

print(age)

Try changing the value of age to 10 and run the code. What happens?

Takeaway: Variables hold information that you can use or change later in your code.
