Python Lesson: Dictionaries

Learn to code from the very beginning

1. Instructions

Dictionaries store information using key-value pairs.

Think of a dictionary like a real dictionary: you look up a word (key) to get a meaning (value).

Type this code into the editor:

ages = {"Alice": 14, "Bob": 16}
print(ages)

Step-by-step:

1. Python creates a dictionary using curly brackets.

2. Each name is a key, and each number is its value.

3. print(ages) shows the whole dictionary, including the curly brackets.

Guided experiment: Add a new name and age inside the dictionary and print it again.

Takeaway: Curly brackets are normal when printing dictionaries. They show all stored key-value pairs.
