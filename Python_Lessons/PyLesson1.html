Python Lesson: Introduction

Learn to code from the very beginning

1. Instructions

Welcome to Python! This is where your coding journey starts.

Python is a programming language. It lets you tell a computer what to do using instructions called code.

Think of Python like a recipe: each line is a step the computer must follow in order.

Type this into the editor:

print("Hello, world!")

What Python does:

1. Looks at the first line.

2. Sees the command print() and knows it should show text.

3. Displays exactly what’s inside the quotation marks in the output area.

Guided experiment: Change the text to your name and run the code. What do you expect to see?

Takeaway: Python reads instructions line by line and follows them exactly. The print() command shows text on the screen.
