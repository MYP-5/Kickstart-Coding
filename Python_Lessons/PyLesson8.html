Python Lesson: Lists

Learn to code from the very beginning

1. Instructions

Lists are like collections of values stored together in one variable.

Example: A list of fruits:

fruits = ["apple", "banana", "cherry"]
print(fruits)

Step-by-step:

1. Python sees the square brackets and stores multiple items in one variable.

2. print(fruits) shows all the items in the list.

Guided experiment: Add another fruit to the list and print it. What happens?

Takeaway: Lists let you store many items together and access them easily.
